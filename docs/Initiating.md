# 🧩 超軽量PMテンプレ（作業なんでも対応版）

これは

* 勉強
* 読書
* 環境構築
* 文章執筆
* リハビリ的な活動

全部に使えます。

---

## 0️⃣ プロジェクト名

```
（例）
・〇〇を試す
・〇〇を整える
・〇〇を理解する
```

👉 **「達成」じゃなく「状態変化」の言葉にする**

---

## 1️⃣ 立ち上げ（意思決定）

### 🎯 ゴール（1行）

```
この作業を通して、どうなっていたらOKか？
```

例：

* 全体像がわかった
* 触るのが怖くなくなった
* 続けるかやめるか判断できた

---

### 🚧 制約（必須）

```
・体調優先
・できない日があっても失敗ではない
・完璧を目指さない
```

👉 **これは毎回コピペでOK**

---

### ✅ 成功の定義（重要）

```
「終わった」ではなく「意味があった」と言える条件
```

例：

* メモが1枚残った
* 次にやる／やらないが決まった

---

## 2️⃣ スコープ（やる／やらない）

### ✅ やること（最大3つ）

```
・〇〇を触る
・〇〇を眺める
・〇〇について1つ質問する
```

---

### ❌ やらないこと（ここがPM）

```
・網羅しない
・比較しすぎない
・完成させない
```

👉 **「やらない」を書いた瞬間、PMが始まる**

---

## 3️⃣ 進め方（最低単位）

### ⏱ 最小アクション

```
・10〜15分
・1ファイル／1ページ／1画面
```

👉 これが **「できたら勝ちライン」**

---

### 📆 期間（雑でOK）

```
・全体：1〜2週間（仮）
・見直し：週1回 5〜10分
```

---

## 4️⃣ チェック（週1・5分）

以下を**ChatGPTにそのまま投げる**だけでOK：

```
・今週やったこと
・しんどかったこと
・来週やらなくていいこと
```

👉 **増やさない。削る。**

---

## 5️⃣ 調整（PMの仕事）

以下のどれかが出たら、計画変更OK：

* 重い
* 面倒
* 触る気が起きない

👉 それは「意志が弱い」のではなく
👉 **計画が現実に合っていないサイン**

---

## 6️⃣ 締め（必須・合否なし）

最後にこれだけ書く：

```
・これは続けたい？ やめたい？
・次にやるなら、何を半分にする？
```

👉 これが **PMBOKでいう「終結」**
👉 次の作業にそのまま流用できる資産

---

# 🔁 このテンプレの強み

* PMBOKの**骨格だけ**を使っている
* アジャイルっぽいが、**スプリントに縛られない**
* 体調・気分・割り込み前提
* 「失敗」という概念がない
